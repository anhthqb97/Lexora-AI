## Summary

Lexora Writing helps Vietnamese learners improve English writing through guided prompts, AI grammar correction, and explain-why feedback.

---

## Goals

| Goal | Metric |
|---|---|
| Submit first essay | ≥70% of activated users |
| Return within 7 days | ≥40% |
| Paid conversion from Writing | ≥5% |

---

## User Stories

| ID | As a | I want | So that |
|---|---|---|---|
| W-01 | Learner | Write on a daily prompt | I build habit |
| W-02 | Learner | See grammar corrections | I learn from mistakes |
| W-03 | Learner | Get scores (grammar, clarity, vocabulary) | I track progress |
| W-04 | Learner | View writing history | I review improvements |

---

## Scope (MVP)

- Text editor (500–2000 chars)
- 10 starter prompts (A2–B2)
- LLM grammar correction + explain-why
- Scores: grammar, clarity, vocabulary (0–100)
- History list + detail view

**Out of scope:** OCR, collaborative editing, IELTS band scoring
